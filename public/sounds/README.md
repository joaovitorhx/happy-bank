# Áudio (opcional)

Para usar sons customizados no jogo, adicione aqui:

- **whoosh.mp3** – tocado ao **enviar** (transferência, pagar banco)
- **coin.mp3** – tocado ao **receber** (receber do banco)

Se os arquivos não existirem, o app usa sons gerados por Web Audio API.

Formato recomendado: MP3, curta duração (0,1–0,3 s), volume moderado.
